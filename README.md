# ❄️ Frost Dashboard

**Frost Dashboard** — это премиальное расширение для Google Chrome, которое полностью преобразует скучную «Новую вкладку» в стильное, высокопроизводительное и невероятно удобное рабочее пространство. 

Разработано с акцентом на **AAA-дизайн**, эстетику Glassmorphism и высокую скорость работы. Идеально подходит для ежедневного использования, работы и даже чтения манги на иностранных языках.

---

## ✨ Главные фишки

* **💎 Премиальный дизайн (Glassmorphism):** Полупрозрачные элементы, глубокое размытие (backdrop-filter), аккуратные тени и уникальный эффект матового стекла.
* **🌌 Интерактивный фон (Aurora):** Живой анимированный фон с эффектом северного сияния, который реагирует на движения мыши (Parallax-эффект).
* **💡 Динамическое освещение:** Карточки и кнопки реагируют на курсор, создавая эффект мягкой неоновой подсветки (GPU-оптимизировано).
* **🔍 Умный поиск (Spotlight style):** Встроенная поисковая строка с автодополнением от Google прямо на лету.
* **🇯🇵 Авто-переводчик картинок (Для Манги):** Уникальное контекстное меню (правый клик по любой картинке в браузере), которое позволяет в один клик перевести текст на картинке через **Яндекс Картинки** (идеально для облачков текста) или **Google Lens**.
* **👋 Умное приветствие:** Дашборд знает, какое сейчас время суток, и здоровается соответствующим образом. Имя пользователя можно редактировать прямо на экране (сохраняется в `localStorage`).
* **📌 Быстрые ссылки:** Кастомная сетка любимых сайтов и боковое меню (в стиле Arc Browser) для мгновенного доступа к закладкам, загрузкам и истории браузера.

## 🛠 Технологии

Проект написан на чистом (Vanilla) стеке без тяжелых фреймворков для мгновенной загрузки:
* **HTML5** — семантическая структура.
* **CSS3** — CSS Variables, Flexbox, Grid, CSS Animations, Backdrop-filter.
* **JavaScript (ES6+)** — логика виджетов, работа с `chrome.*` API, Service Workers.
* **Chrome Extension API (Manifest V3)** — современный стандарт расширений, обход CORS для автодополнения и интеграция с Context Menu.

## 📦 Как установить себе

Так как проект пока не опубликован в Chrome Web Store, вы можете установить его в режиме разработчика за 1 минуту:

1. Скачайте этот репозиторий (кнопка `Code` -> `Download ZIP`) и распакуйте его в любую папку.
2. Откройте браузер Google Chrome и перейдите по адресу: `chrome://extensions/`
3. В правом верхнем углу включите тумблер **"Режим разработчика"** (Developer mode).
4. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked) в левом верхнем углу.
5. Выберите папку, в которую вы распаковали архив.
6. **Готово!** Откройте новую вкладку и наслаждайтесь!
